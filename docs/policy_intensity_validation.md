# 强度系统验证

## 正式门槛

- 动作识别 macro-F1 ≥ 0.85
- 工具多标签 macro-F1 ≥ 0.85
- 方向 macro-F1 ≥ 0.90
- 数值抽取准确率和旧新值配对准确率 ≥ 0.95
- 双人标注 Kappa ≥ 0.75
- 综合指数与专家评分 Spearman ≥ 0.80

切分必须同时报告随机分层、政策实体/转载族分组、城市留出和时间留出。延迟、Token、费用、缓存命中和无证据率与分类指标并列报告。

## 当前结果

本地 100 篇官方状态记录试点提取 858 个候选动作，其中 607 个满足保守的完整官方文本规则，251 个为 provisional；提取 336 个数值候选，仅 7 个形成确定性旧值/新值配对。该结果是管线运行计数，不是准确率。人工金标准为 0，传统模型和 Transformer 均按设计返回 `blocked_missing_gold`，因此正式门槛全部未评估，`research_ready=false`。

GLM 受限试点中，初始结果因字符偏移不匹配被拒绝；加入“证据必须在原文唯一出现才允许程序重定位”后，1 个动作的抽取和独立复核成功。已知 Token 至少 2,533，完整 Token 和费用不可用；单样本不能计算幻觉率或准确率。
