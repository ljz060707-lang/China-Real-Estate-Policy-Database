# 政策强度编码手册

机器可读版本位于 `data/reference/policy_intensity_codebook.yaml`。

| 代码 | 维度 | 0 | 1 | 2 | 3 |
|---|---|---|---|---|---|
| D1 | 目标明确性 | 无目标 | 原则目标 | 明确对象或结果 | 对象及可核验目标 |
| D2 | 范围 | 不明 | 宽泛 | 明确地区或对象 | 地区、对象、住房范围完整 |
| D3 | 整合协调 | 无 | 提及协同 | 明确联合主体 | 主体分工和联动机制 |
| D4 | 资源承诺 | 无 | 原则支持 | 明确资源类型 | 明确数量、额度或配置 |
| D5 | 实施程序 | 无 | 仅提出实施 | 明确步骤或主体 | 步骤、主体、期限完整 |
| D6 | 监测问责 | 无 | 原则监督 | 明确监测报告 | 考核问责和频率 |
| D7 | 约束力 | 描述 | 鼓励推动 | 应当需要 | 必须不得严格执行 |
| D8 | 校准幅度 | 不适用时 NA | 确定性连续值 | 确定性连续值 | 确定性连续值 |

`NA` 与 0 不同。没有跨部门协调需求的单一动作，D3 为 NA；明确需要协调但文本没有安排时，D3 才可能为 0。所有非 NA 自动标签需要证据片段、偏移、方法、置信度和复核状态。
