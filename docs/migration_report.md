# 迁移报告

本报告由实际迁移和验证结果补充。28 个工作表全部进入单元格级 Staging；T1 为 1 行表头加 3011 条非空记录。T2 是 306 列、627 个合并区域的复杂矩阵，共保存 4,151 个非空或合并单元格；其中按城市分块和可解释标题确定性转换出 1,794 条 `city_policy_rules`，其余标题、目录、合并占位或语义不确定单元格不猜测。T4 第 9–2118 行正好是 2,110 行人工需求侧编码（此前“约 2110”即来自该口径），生成 10,663 条 `policy_features`，其中 8,631 条通过“城市+日期唯一匹配”关联主记录，并安全提取 1,166 条定量措施；该表 42,477 个公式及图表辅助列不作为原始事实。

主目录缺失、日期范围、重复与分类质量以 `data/staging/validation_report.json` 为准。专题表按工作表来源迁入 records；简版、横版和汇总展示保留于 Staging，未作为独立基础政策事实重复导入。另生成 298 条城市能级属性、3,330 条来源文档关系和 97 条白名单/PSL 项目事件。T4 尚有 2,032 条特征因城市+日期对应多条或无主目录候选而未强行关联，进入人工审核范围。

## 七大政策库整理结果

按部门职责与研究用途新增七大库、34 个细分类。程序复核结果为：28/28 个工作表均有明确资料归属，91,793/91,793 个 Staging 单元格均进入统一查询视图和单元格目录；3,568/3,568 条 Curated 记录至少有一个政策库归属，共生成 18,974 条多标签关系。未映射工作表、未编目单元格和未归库记录均为 0。

记录覆盖数为：党中央国务院政策 458、住房城乡建设部门政策 1,744、金融监管部门政策 1,693、自然资源部门政策 493、地方政府房地产政策 3,128、房地产风险支持专项库 528、政策跟踪与统计库 3,150。上述数字相互重叠，不能求和作为政策总量。

信息完整保留不代表自动分类已全部人工确认。来源工作表映射为确定性资料归属；关键词、地域规则产生的关系保留证据和置信度，审核状态为 `unreviewed`。T2 语义不确定单元格、T4 未关联特征及既有低置信度任务继续进入人工审核中心，不进行猜测。
