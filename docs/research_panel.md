# 105城市研究面板

范围来自国务院第七次全国人口普查领导小组办公室编制的《2020中国人口普查分县资料》。财政部转载说明该口径共有105个大城市：7个超大城市、14个特大城市、14个Ⅰ型大城市、70个Ⅱ型大城市。

参考：[财政部安徽监管局转载的105个大城市说明](https://ah.mof.gov.cn/lianzhengjianshe/202211/t20221114_3851397.htm)。名单版本为 `2020-census-v1`，统计时点为2020-11-01。

视图：

- `v_policy_105_cities`：2018年至今政策—适用城市长表；
- `v_city_month_policy_panel_105`：105城市完整月份网格；
- `v_city_year_policy_panel_105`：年度聚合。

城市能级是独立分析属性。101个城市可直接连接原“能级划分”表；昆山、义乌、慈溪、晋江为县级大城市，原298城市表没有同名能级，因此保持空值，不继承上级城市等级。

`data_completeness_score` 是透明的数据完整性指标：官方来源占60%、正文可用占20%、来源链接可用占20%。它不是政策效果或因果指标。政策强度首先使用可修改的确定性规则，GLM方向仅作辅助。
