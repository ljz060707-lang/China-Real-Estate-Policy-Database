# AI 分类与独立复核方法

流程为：确定性相关性筛选 → 动作拆分 → 分类模型 → 独立复核模型 → 程序路由。模型只能输出
结构化字段和逐字段原文证据；最终自动通过由程序决定。

自动通过必须同时满足：一级分类唯一、二级分类从属于一级分类、证据可在源文唯一定位、两轮
分类和方向一致、适用地区无冲突、数值规则无冲突。任何无证据结果都不得自动通过。

人工托底只接收：模型冲突、证据缺失、地区冲突、重大修订、数值新旧值冲突、疑似误合并和
模型连续失败。人工原判断、AI 原判断、最终裁决、模型/Prompt/Schema/Codebook 版本必须并存。

当前尚无 300–500 条完成裁决的分层人工金标准，因此 macro-F1 与研究就绪状态不能报告为达标。
